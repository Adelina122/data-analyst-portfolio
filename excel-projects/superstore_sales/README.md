## Superstore Sales Analysis (Excel Project)
### Project Overview

This project is an end-to-end Excel data analysis based on the popular Superstore sales dataset.
The goal of the project is to demonstrate how raw, messy business data can be cleaned, analyzed, and turned into clear insights using Excel only.

The project reflects a realistic junior data analyst workflow: working with imperfect data, structuring it properly, and answering business-relevant questions through analysis and visualization.

### Objectives

- Clean and standardize raw sales data
- Analyze sales and profit performance
- Identify trends across categories, regions, and customer segments
- Create a clear and easy-to-understand Excel dashboard for decision-making
