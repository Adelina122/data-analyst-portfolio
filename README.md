# data-analyst-portfolio
A collection of beginner-friendly data analysis projects using Excel, SQL, and Power BI, focused on data cleaning, reporting, and turning data into clear insights for decision-making.

## Project Activity & Impact Analysis (Excel)

**Tools:** Excel  
**Skills:** Data cleaning, pivot tables, KPIs, data visualization, insights

### Objective
Analyze project participation and cost data to identify trends and support decision-making.

### Key Tasks
- Cleaned and prepared raw data
- Created pivot tables and KPIs
- Built charts to visualize trends
- Provided actionable insights

### Key Insights
- Participation increased over time
- Some projects achieved higher impact at lower cost
- Recommendation: focus resources on high-impact projects
